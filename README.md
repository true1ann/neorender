# NeoRender

NeoRender is a lightweight rendering engine designed for web-based JavaScript projects. It aims to provide a React-like experience without the overhead of React itself, and focused on optimization. _(soon)_

## Key Features:

* Lightweight: Minimal footprint with no unnecessary bloat.
* Dependency-Free: Built to operate without external libraries, reducing potential conflicts and improving load times. _(hopefully will stay like this)_
* Declarative Syntax: Offers a clear and intuitive way to define UI components.
* React-like: Supports reactive programming principles to automatically update the UI in response to data changes. _(soon)_
